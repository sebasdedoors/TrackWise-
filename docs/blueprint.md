# **App Name**: TrackWise

## Core Features:

- Task Creation: Create tasks with title, category (Work/School/Personal/Health), priority (Low/Medium/High), and due date (defaults to today).
- Task Management: Basic task management functionalities: edit and delete tasks.
- Task Completion: Allow users to mark tasks as completed.
- "Today" View: A view that displays the list of tasks for the current day.
- Task Counter: Display a counter of pending and completed tasks in the "Today" view.
- Task Filtering: Filter tasks by category or priority in the "Today" view.
- Quick Checklist: A quick checklist (max 5 items) for adding simple tasks or sub-tasks to the Daily Tracker.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6), representing focus and organization.
- Background color: Light grey (#FFFFFF), for a clean and unobtrusive backdrop.
- Accent color: Soft teal (#4DB6AC), used for highlights and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern look with a little warmth and personality.
- Simple, clear icons to represent task categories and priority levels.
- Clean, card-based layout for tasks, promoting easy scanning and interaction.
- Subtle animations on task completion and transitions between views.
